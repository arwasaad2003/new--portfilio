Pose Estimation for Real-Time Repetition Counter using Mediapipe
Goal

Build a real-time pose estimation system that detects human body keypoints from webcam input and tracks elbow joint angles to count exercise repetitions.

Solution

Utilized the Mediapipe Pose model integrated with OpenCV to access webcam footage and extract landmark coordinates in real time. The system calculates the elbow angle dynamically by monitoring the position of the shoulder, elbow, and wrist joints, and applies simple logic to detect "up" and "down" stages in a movement. Each completed movement triggers a count, simulating real-time exercise repetition tracking..

Impact

Delivered a lightweight yet functional fitness tracking prototype, suitable for personal workout monitoring and further development into full-scale fitness assistant applications

Face Mask Detection with Deep Learning (CNN)

Goal

Build an image-based deep learning model to detect whether individuals in photos are wearing face masks or not, using Convolutional Neural Networks (CNN).

Solution

Utilized a labeled dataset of face images categorized as "with mask" and "without mask". Preprocessed the images by resizing, normalizing, and augmenting the data. Designed and trained a **CNN model** using **TensorFlow** and **Keras**, achieving strong classification performance. Evaluation was done on a separate validation set with visual plots for accuracy and loss tracking

Impact

The model accurately classified face mask usage in still images, demonstrating effective use of CNNs for binary image classification. This system can be used in static analysis settings such as photo review tools or offline monitoring solutions

ML Titanic-Survivals-Prediction

Goal

Build a predictive model to determine the likelihood of a passenger surviving the Titanic shipwreck, using demographic and travel-related features from the Kaggle Titanic dataset.

Solution

Performed in-depth exploratory data analysis (EDA) to understand feature correlations and missing values. Applied data cleaning and feature engineering to enhance model input. Trained a **Logistic Regression** model using **Scikit-learn**, and evaluated its performance using accuracy, confusion matrix, and cross-validation techniques. Visualized insights with **Matplotlib** and **Seaborn** to communicate survival patterns based on factors like gender, passenger class, and age

Impact

Gained hands-on experience in the end-to-end machine learning workflow, from data preprocessing to model deployment. Built a reliable and interpretable model capable of predicting survival with strong accuracy. Developed valuable skills in EDA, classification models, and feature importance analysis.
